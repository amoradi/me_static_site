## Local Dev

Built with [Zola.](https://www.getzola.org/)

```bash
# Install zola
$ brew install zola

# Build
$ zola build

# Build and serve site locally
$ zola serve
```

## Publish to Gihub Pages

```bash
$ bash publish_www.sh
```
